01

Бэктестер прогоняет внутридневную стратегию по минутным котировкам одного инструмента и считает результат.
Поддерживаются два режима использования:
— Одиночный тест — один набор параметров, полный набор метрик и анализ капитала.
— Оптимизация — перебор сетки параметров (диапазоны/перечисления) с параллельным расчётом, грубыми метриками и записью сводки в CSV.

Торговый робот по API создаёт ордера и входит в сделки.
Торгует фьючерсами по стратегиям, найденным через тестер.
Поддерживает работу с API брокеров:
IBKR, Bybit, Финам, АЛОР.

GUI для удобства и визуализации.
Весь функционал бэктестера и торговли в одном приложении.

Kronos-Lab

04

Исследовательский стенд для систематической торговли. Генеративная модель строит ансамбли из 500+ вероятных 24-часовых траекторий на основе рыночного контекста.

Бэктестер с резюмированием, авто-адаптацией под GPU и разделением генерации/анализа: ансамбли считаются один раз на GPU, стратегии перебираются на CPU бесплатно.

Сохраняет полный контракт данных: траектории, контекстные фичи (волатильность, тренд, z-score), реальный исход. Оценка сигналов по независимому суточному срезу — без перекрытия сделок.

Использует нейросеть Kronos-Tokenizer-base для генерации ансамблей.

Kharon Messenger

02

Минималистичный зашифрованный мессенджер для небольших команд.

v2.0.0 — Reception Hours Edition
Клиент больше не обязан висеть в сети 24/7. Зашёл → Забрал → Ушёл. Следов нет.

Reception Mode: LIVE, PULSE_5/15/30/60/240/360/720/1440 (окно связи раз в N минут), SILENT. Режим собеседника виден в чате.

Динамический TTL на сервере: сообщение живёт 2 минуты + интервал PULSE получателя. PULSE_1440 → TTL 26 часов.

Счётчик непрочитанных в списке контактов, сбрасывается при открытии чата. PULSE-уведомления с вибрацией, автозапуск после перезагрузки.

4 темы оформления: Princess ♡ (дефолтная), Terminal Dark >_!, Terminal Light о_0, Shadow ☠.

Шифрование: E2E X25519 + XSalsa20-Poly1305, WebSocket с certificate pinning, Android Keystore, Room + SQLCipher, FLAG_SECURE.

AIM — All In Me

03

Анонимная платформа (Telegram Mini App + Web) для создания глубокого психологического профиля. Проект включает полностью спроектированный бэкенд, базу данных и клиентскую часть.

Ключевые возможности и архитектура:
• 4 типа записей: Сны, Воспоминания, Мысли, Планы с гибкой системой связей.
• 27 научно обоснованных эмоций (UC Berkeley, 2017) с градацией интенсивности.
• Учет контекста: привязка к обстоятельствам (погода, фаза луны, глобальные события) и физическому состоянию (геолокация, здоровье, энергия).
• RPG-механика: система навыков и прогресса, привязанная к записям.

Техническая реализация:
• Полноценный бэкенд на Express (Node.js).
• База данных PostgreSQL с расширением PostGIS для геоданных.
• Клиентская часть на TypeScript / React.
• Отдельный микросервис на Python для AI-функций (Jungian-аналитика, генерация изображений и видео).
• Архитектура, построенная на принципах ООП и четком разделении ответственности.
